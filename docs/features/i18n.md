# i18n & Localization

## Supported Locales

| Locale | Language | Dictionary | Size |
|--------|----------|------------|------|
| en-GB | British English | CSW (via word-list) | ~274k words |
| en-US | American English | NWL (via word-list) | ~274k words |
| de | German | Standard (via an-array-of-german-words) | ~117k words |

## URL Architecture

- `/[locale]/` prefix for all pages
- `/` redirects to browser's preferred locale
- Locale aliases: `de-DE` → `de`, `de-AT` → `de`, `en` → `en-US`

## Translation Files

Located in `public/locales/`:
- `en-GB.json`
- `en-US.json`
- `de.json`

Keys follow the format: `section.key` (e.g., `letters.title`, `numbers.submit`)

## Dictionary Loading

- English: `word-list` npm package (reads words.txt from package)
- German: `an-array-of-german-words` npm package (JSON array)
- Loaded lazily on first access, cached in memory
- `loadDictionary(locale)` returns `Set<string>` of uppercase words

## Umlaut Handling

German special characters are mapped at validation time:
- `normalizeGermanWord()` expands umlauts before character frequency check
- Applied to both tiles and player input
- Board displays original letters (not expanded)
